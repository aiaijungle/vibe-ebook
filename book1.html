혼자 사업을 하면 마케팅이 제일 무겁다. 블로그도 써야 하고, 유튜브도 올려야 하고, 고객도 관리해야 한다. 직원이 없으면 내가 다 해야 한다.

📌 이 책에서 다루는 것
블로그 자동화 — 글을 직접 쓰지 않아도 매일 포스트가 올라가는 구조

이론 최소. 실전 최대. 오늘 바로 써먹을 수 있는 것만 담았다.

1장. 블로그 자동화

검색에서 들어온 사람이 가장 구매 확률이 높다. 광고는 내가 먼저 찾아가지만, 블로그는 고객이 먼저 찾아온다. 이미 그 문제에 관심이 있는 사람이다. 설득이 반쯤 된 상태다.

문제는 직접 쓰면 지친다는 거다. 주 3편, 3개월. 그러다 멈춘다. 안 올라온 블로그는 검색에 잡히지 않는다.

🔥 핵심 원리
블로그는 꾸준함이 아니라 시스템이 답이다.
사람이 꾸준한 게 아니라, 봇이 꾸준하게 만들어야 한다.

구조

트렌드 키워드 수집 → AI 글 생성 → 자동 발행
      ↓                   ↓              ↓
  매일 오전           Claude API     GitHub Pages

매일 아침, 봇이 오늘의 키워드를 가져온다. AI가 글을 쓴다. 자동으로 올라간다. 당신은 아무것도 안 해도 된다.

지금 당장 해볼 것

코드가 없어도 된다. Claude 웹 버전에 이렇게 입력해라.

"내 블로그 주제는 [본인 분야]. 오늘 쓸 블로그 제목 5개 추천해줘. SEO에 유리한 키워드 포함해서."

마음에 드는 제목 고르고:

"이 제목으로 800자 블로그 포스트 써줘. 소제목 2개 넣어줘."

하루 10분. 포스트 하나. 자동화는 이걸 사람 없이 돌리는 거다.

💡 주의: AI 글은 꼭 첫 단락 읽어보고 올려라. 사실 오류가 있을 수 있다. 방향만 잡아주면 된다.

2장. 유튜브 자동화

유튜브는 얼굴을 비춰야 한다고 생각한다. 아니다. 얼굴 없이 수십만 구독자를 가진 채널들이 있다. AI 음성과 배경 영상만으로 운영한다.

쇼츠 자동 제작

쇼츠(60초 이내 세로 영상)는 알고리즘이 새 채널을 밀어주는 포맷이다. 구독자가 없어도 조회수가 나온다. 이걸 자동화한다.

주제 입력
  → Claude가 대본 작성
  → ElevenLabs가 음성 변환 (무료 플랜: 월 10분)
  → Pexels에서 배경 영상 다운로드 (무료)
  → 편집앱으로 합성
  → 유튜브 업로드
📌 비용 합계
Claude 웹: 무료 | ElevenLabs 기본: 무료 | Pexels: 무료
총 비용 0원. 시간 30분. 영상 1편 완성.

지금 당장 해볼 것

Claude에 입력:

"60초 유튜브 쇼츠 대본 써줘. 주제: [오늘 알게 된 것 하나]. 첫 3초가 훅이 되게. 말하듯이 써줘."

대본 복사 → ElevenLabs에 붙여넣기 → 음성 다운로드 → 배경 영상 합치기 → 업로드. 오늘 안에 첫 쇼츠가 올라간다.

3장. 연결하면 달라진다

블로그와 유튜브를 따로 돌리는 것보다 연결하면 효과가 커진다.

🔥 자동화 루프
블로그 포스트 작성 → 같은 내용으로 쇼츠 대본 생성 → 쇼츠 업로드
→ 영상 설명란에 블로그 링크 → 블로그에 유튜브 영상 삽입

하나의 콘텐츠가 두 채널을 동시에 채운다.

블로그로 검색 유입. 유튜브로 알고리즘 유입. 두 채널이 서로를 밀어준다. 이게 1인 기업가가 마케팅팀 없이 할 수 있는 가장 효율적인 구조다.

이건 시작일 뿐이다

블로그와 유튜브에서 들어온 사람을 이메일로 전환하고,
이메일 시퀀스로 자동 판매하고,
고객 DB가 자는 동안 쌓이는 구조.
